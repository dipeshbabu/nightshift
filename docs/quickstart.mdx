## Install the toolchain

After building or installing Nightshift, prepare the agent and it's environment with:

```bash
nightshift install --prefix ~/my-agent
```

## Launch the Agent

### Run Opencode with prefix as working directory
```bash
nightshift run 
```
### Run Ralph from prompt file
```bash
nightshift run --ralph --prompt task.txt
```

### Running Ralph server

```bash
nightshift run --ralph serve 
```

### Running Ralph with Jobs TUI
```bash
nightshift run --ralph serve --run-nightshift-tui
```
