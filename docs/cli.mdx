---

Nightshift exposes three main commands: `install`, `run`, and `attach`.

## install

Download and install tools into a prefix directory.

```bash
nightshift install --prefix ~/.nightshift
```

| Option | Description |
|--------|-------------|
| `--prefix` | Directory to install tools into (required) |

## run

Launch OpenCode with an isolated PATH pointing to the installed toolchain.

```bash
nightshift run --prefix ~/.nightshift [options] [-- opencode-args]
```

| Option | Description |
|--------|-------------|
| `--prefix` | Prefix directory containing installed tools (required) |
| `--run-nightshift-tui` | Start OpenCode in server mode and attach the Nightshift TUI |
| `--` | Pass remaining arguments directly to OpenCode |

### Examples

```bash
# Run OpenCode with isolated tools
nightshift run --prefix ~/.nightshift

# Run with the Nightshift TUI
nightshift run --prefix ~/.nightshift --run-nightshift-tui

# Pass arguments to OpenCode
nightshift run --prefix ~/.nightshift -- --continue
```

## attach

Connect the Nightshift TUI to an existing OpenCode server.

```bash
nightshift attach <url> [options]
```

| Option | Description |
|--------|-------------|
| `--session` | Session ID to attach to |

### Example

```bash
nightshift attach http://127.0.0.1:4096 --session <session-id>
```
